# Spotify Clone 🎵

A simple **Spotify UI Clone** built using **HTML and CSS**.
This project focuses on recreating the layout and design of the Spotify web interface.

## 🚀 Features

* Responsive Spotify-like layout
* Sidebar navigation menu
* Music cards section
* Horizontally scrollable playlists
* Clean and modern UI

## 🛠️ Built With

* **HTML5**
* **CSS3**
* Flexbox
* Basic responsive design

## 📂 Project Structure

```
spotify-clone/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## 🎯 Purpose of the Project

This project was created to practice:

* HTML page structuring
* CSS layouts using **Flexbox**
* Creating responsive UI components
* Building real-world UI clones

## 💻 How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser
